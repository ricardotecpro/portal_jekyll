$ python script.py

2. Diagramas (Mermaid.js)

Podemos usar diagramas para explicar conceitos complexos como herança de classes.

Exemplo de Herança

classDiagram
    Animal <|-- Cachorro
    Animal <|-- Gato
    class Animal {
        +String nome
        +fazerSom()
    }
    class Cachorro {
        +latir()
    }
    class Gato {
        +miar()
    }
