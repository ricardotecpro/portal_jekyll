🎓 Módulo 0: Configuração do Ambiente

Objetivo: Preparar sua máquina para o desenvolvimento Full Stack com Java, React e React Native.

1. Pré-requisitos

Antes de começar a codificar, precisamos instalar as ferramentas base.

1.1. Java Development Kit (JDK) 21

O Java é a base do nosso Backend.

  • Download: Adoptium Temurin 21 (LTS)
  • Verificação: Abra o terminal e digite: 
    java -version

    Deve aparecer algo como “openjdk version 21…”

1.2. Node.js (LTS)

O Node.js é o motor que roda o nosso Frontend e Mobile.

  • Download: Node.js (Versão LTS)
  • Verificação: 
    node -v
npm -v

1.3. Visual Studio Code (VS Code)

Nosso editor de código principal.

  • Download: VS Code
  • Extensões Recomendadas:
    • Extension Pack for Java (Microsoft)
    • Spring Boot Tools (VMware)
    • ES7+ React/Redux/React-Native snippets (dsznajder)
    • Lombok Annotations Support

1.4. Postman (Opcional)

Para testar nossa API antes de ter o Frontend pronto.

  • Download: Postman

2. Configurando o Git (Opcional mas Recomendado)

Se você quiser salvar seu código no GitHub:

  1. Instale o Git.
  2. Configure seu usuário: 
    git config --global user.name "Seu Nome"
git config --global user.email "seu@email.com"

✅ Checkpoint

Se você digitou java -version e node -v no terminal e viu as versões instaladas, parabéns! Sua máquina está pronta para o Módulo 1.
