## 📊 Como Funcionam os Quizzes

### Interatividade
Cada quiz possui questões de múltipla escolha com feedback imediato. Clique em uma opção para ver se está correta e receber uma explicação.

### Feedback Educativo
Todas as respostas incluem explicações detalhadas para ajudar no aprendizado, mesmo quando você acerta!

### Sem Limite de Tentativas
Você pode refazer os quizzes quantas vezes quiser para fixar o conhecimento.
